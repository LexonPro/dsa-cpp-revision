# 📦 DSA C++ Revision
 
A clean, modern, and visually impressive **Data Structures & Algorithms revision website** built with pure HTML, CSS, and JavaScript — designed for placement and interview preparation.
 
🌐 **Live Site → [lexonpro.github.io/dsa-cpp-revision](https://lexonpro.github.io/dsa-cpp-revision/)**
 
---
 
## ✨ Features
 
- 🎨 **Premium dark UI** with gradient accents and smooth animations
- 📌 **Fixed sidebar** with active section highlighting
- 📊 **Complexity tables** for every operation
- 🔷 **8 algorithmic patterns** with collapsible cards
- 💻 **Syntax-highlighted C++ code** for every pattern
- ⎘ **Copy code button** on every snippet
- 📱 **Fully responsive** — works on mobile, tablet, and desktop
- ⚡ **Read progress bar** at the top
- 🚀 **Zero dependencies** — single HTML file, no npm, no frameworks

## 🚀 Getting Started Locally
 
No setup needed. Just open the file:
 
```bash
# Clone the repo
git clone https://github.com/LexonPro/dsa-cpp-revision.git
 
# Open in browser
cd dsa-cpp-revision
start index.html       # Windows
open index.html        # macOS
xdg-open index.html    # Linux
```
